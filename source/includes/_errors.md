# HTTP Status Codes

The Wufoo API uses the following status codes:

Error Code | Meaning
---------- | -------
400 | Bad Request
401 | Unauthorized -- Your API key is wrong
403 | Forbidden 
404 | Not Found 
405 | Method Not Allowed 
406 | Not Acceptable 
409 |
429 | Too Many Requests 
500 | Internal Server Error 
503 | Service Unavailable
